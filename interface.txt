1. 厂区设置 - 最后增加二级菜单：厂区类型默认图片设置
	NOTE: 分区没有图片（即：area.image为空）时根据area.type获取对应的level.image
				保存分区信息时，如果使用的是level.image,那么发保存请求时area.image也为空
	1.1 增加接口
		路径：	/level/addLevel
		输入参数：
			level: int,与已有level不重复
			name：string，用户设置
			image：String，用户上传自己图片
			userid：当前用户
		输出参数：
			status：0/400
	1.2 修改接口
		路径：	/level/modifyLevel
		输入参数：
			id:int,已有的id
			level: int,不能修改，仅仅显示
			name：string，用户设置
			image：String，用户上传自己图片
			userid：当前用户
		输出参数：
			status：0/400
	1.3 查询接口
		路径：	/level/levels
		输入参数：无
		输出参数：
			status：0/400
			data：[
				{
					id:int, id
					level: int,层级
					name：string，层级名称
					image：String，图片
				}
			]
	1.4 删除接口
		路径：	/level/deleteLevel
		输入参数：
			id:int,已有的id
			userid：当前用户
		输出参数：
			status：0/400
